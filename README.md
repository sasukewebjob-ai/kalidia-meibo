# KALIDIA名簿

バドミントンサークル「KALIDIA」の出欠を記録するためのシンプルな名簿票Webアプリ。
スマホのChromeで動かして、練習日ごとにタップで ○ を付けられる。

## 使い方

1. ブラウザで `index.html` を開く（GitHub Pagesで公開してスマホからアクセスもOK）
2. 右上の「＋日付」ボタンで日付列を追加
3. 日付ヘッダーをタップで日付を編集・列を削除
4. 名前と日付が交差するセルをタップで ○ のオン/オフ
5. データは自動で端末内に保存される

## 特徴

- **単一HTMLファイル** — ビルド不要、サーバー不要
- **オフライン動作** — 一度開けばネットなしでも使える
- **自動保存** — `localStorage` に即時保存、リロードしても消えない
- **横スクロール対応** — 氏名列は固定、日付は好きなだけ追加
- **JSON入出力** — エクスポート／インポートで別端末へデータ移行可能

## メニュー

右上の「⋮」ボタンから：

- **📥 JSONエクスポート** — 全データを `kalidia-roster-YYYY-MM-DD.json` でダウンロード
- **📤 JSONインポート** — エクスポートしたJSONを読み込んで復元
- **🗑 出席データをリセット** — 日付列と出席データを全削除（メンバーは残る）

## メンバーの変更

`members.txt` を編集して：

```
python update_members.py
```

で `index.html` に反映される。詳しくは [CLAUDE.md](CLAUDE.md) 参照。

## GitHub Pagesで公開する

1. このフォルダをGitHubにpush
2. Repository → Settings → Pages
3. Source: `main` branch / `/ (root)` を選択
4. 発行されたURL（`https://<user>.github.io/<repo>/`）をスマホのChromeで開く
5. Chromeのメニュー →「ホーム画面に追加」でアプリ風に使える

## データの扱い

データは**開いた端末のブラウザ内（localStorage）にだけ**保存される。
- 別の端末では別のデータになる（同期はしない）
- ブラウザのデータを消すと一緒に消える
- バックアップはJSONエクスポートで
